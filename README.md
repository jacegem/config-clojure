# config-clojure


## install

```
# one-off to install clj-new as a tool:
clojure -Ttools install com.github.seancorfield/clj-new '{:git/tag "v1.2.381"}' :as clj-new
```


## clone
```
git clone https://github.com/jacegem/config-clojure.git ~/.config/clojure
```
